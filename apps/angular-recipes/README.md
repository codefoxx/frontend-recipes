# Angular Recipes

Angular version of the Frontend Recipes app.

This app mirrors the React scaffolding so new recipes can be added with the same mental model:

- in-memory recipe definitions
- route-based recipe pages
- recipe registry for demos
- shared layout and section components
- lightweight markdown rendering for recipe content

## Scripts

- `npm run dev` starts the Angular dev server
- `npm run build` builds the app
- `npm run typecheck` runs TypeScript type checking
- `npm run format` formats the project
