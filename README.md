# Frontend Recipes

Practical patterns for recurring frontend problems.

The repository contains small executable examples ("recipes") demonstrating solutions to common frontend challenges such as:

- debounced search
- protected routes
- permission-based UI

The first implementations are written in **React + TypeScript**.  
Future versions may include **Angular implementations** for selected recipes.

## Tooling

The project uses a modern frontend tooling stack:

- **Vite** for fast development builds
- **TypeScript** for type safety
- **ESLint (flat config)** for code quality
- **Prettier** for automatic formatting

Code is automatically formatted on save in VS Code using Prettier.
